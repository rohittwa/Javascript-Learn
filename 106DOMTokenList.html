DOM token list

The DOM token list interface represents a set of space-separated tokens. Such a set is returned by Element.classList, HTMLLinkElement.relList, HTMLAreaElement.relList, HTMLframeElement.sandbox, or HTMLOutputElement.htmlFor. etc. It is indexed beginning with 0 as with Javascript Array objects. DOMTokenList is always case-sensitive.

Properties in DOMTokenList :-
  • length
  • value
Methods in DOMTokenList :-
  • item() : It returns a class which is at specified index.
  • add()
  • remove()
  • replace()
  • contains()
  • toggle()
