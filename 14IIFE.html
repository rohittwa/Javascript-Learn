IIFE

IIFE (Immediately Invoked Function Expression) is a javascript function that runs as soon as it is defined.
It is a design pattern which is also known as Self-Executing Anonymous function with lixical scope enclosed within the Grouping Operator(). This prevents accessing variables within the IIFE idiom as well as polluting the global scope.
The second part is creating the immediately executing function expression(), through which the javascript engine will directly interpret the function.
Ex:-
(function() {document.write('Geekyshows');})(); (function(a, b) {document.write(a + "" + b);})(10, 20);
