Events

HTML events are "things" that happen to HTML elements.
When JavaScript is used in HTML pages, JavaScript can "react" on these events.

HTML Events

An HTML event can be something the browser does, or something a user does.

Here are some examples of HTML events:

  • An HTML web page has finished loading
  • An HTML input field was changed
  • An HTML button was clicked

Often, when events happen, you may want to do something.
JavaScript lets you execute code when events are detected.
HTML allows event handler attributes, with JavaScript code, to be added to HTML elements.

We can use Events with DOM by two methods :-
1. document.getElementById("div1").event_name = function_name;
2. document.getElementById("div2").addEventListener("event_name", function_name, useCapture);
The event_name must be used without the word 'on'. The useCapture value can be true or false. To remove an event from an element:-
document.getElementById("div2").removeEventListener("event_name", function_name, useCapture);
The arguments in removeEventListener() method must be the same as in addEventListener() method.

In the following example, an onclick attribute (with code), is added to a <button> element:

<button onclick="document.getElementById('demo').innerHTML = Date()">The time is?</button>

Common HTML Events

Here is a list of some common HTML events:

Event Triggers when
onchange An HTML element has been changed and the focus is not at the element
onclick The user clicks an HTML element
onblur The user removes the focus from the element
onkeydown The user pushes a keyboard key
onkeypress The event occurs when the user pressed a key. It only works with printable characters
onkeyup The event occurs when the key is released.
onselect The user selects the text inside the textbox.
oninvalid The user has input the wrong value or has not given a value.
onkeyup The event occurs when the user releases a key
oninput The event occurs when an element gets user input
oncontextmenu The event occurs when the user right-clicks on an element to open a context menu
ondblclick The event occurs when the user double-clicks on an element
onmouseover The user moves the mouse over an HTML element
onmouseout The user moves the mouse away from an HTML element
onmousedown The event occurs when the user presses a mouse left button over an element
onmouseenter The event occurs when the pointer is moved onto an element. No bubble
onmouseleave The event occurs when the pointer is moved out of an element. No bubble
onmousemove The event occurs when the pointer is moving while it is over an element
onmouseover The event occurs when the pointer is moved onto an element, or onto one of its children
onmouseout The event occurs when a user moves the mouse pointer out of an element, or out of one of its children
onmouseup The event occurs when a user releases a mouse button over an element
onwheel The event occurs when the mouse wheel rolls up or down over an element
offline The event occurs when the browser starts to work offline
online The event occurs when the browser starts to work online
onsubmit The event occurs when a form is submitted
onload The event occurs when a page has finished loading
onunload
onresize The event occurs when the window is resized
onscroll The event occurs when the document is scrolled
onfocus The event occurs when an element gains focus, such as selecting a form field. No Bubble
onblur The event occurs when an element loses focus, such as moving away from a form field. No Bubble
focusin The event occurs when an element is just about to gain focus
focusout The event occurs when an element is just about to lose focus
