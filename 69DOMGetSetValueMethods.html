DOM get set value methods

Q. What we can get with DOM?
Ans:
  • HTML
  • Text
  • Attribute


Q. DOM get methods?
Ans:
  • innerText: gives all the text.
  • innerHTML : gives all the HTML tags and their text inside the element
  • value : gives the value of a text box
  • textContent : gives the text of the element.
  • getAttribute : gets the attribute name.
  • getAttributeNode : get the attribute name and value.
  • attributes : gets all the attibutes in an element.


Q. DOM set methods?
Ans:
  • innerText: sets all the text.
  • innerHTML : sets all the HTML tags and their text inside the element
  • value : sets the value of a text box
  • textContent : sets the text of the element.
  • setAttribute(attributes_name, value)
  • removeAttribute
