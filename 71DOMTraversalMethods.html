DOM Traversal Methods

  1. parentNode : It returns the parent node of an element. If the parent node doesn't exist, it returns document.
  2. parentElement : It returns the parent node of an element.
  3. children : Returns the tags which are child of the accessed tag.
  4. childNodes : Returns the tags, comments and 'text' if normal text is written including enter, space and tab button which are child of the accessed tag.
  5. firstElementChild : Returns the first child tag which is in the accessed element.
  6. lastElementChild : Returns the last child tag which is in the accessed element.
  7. firstChild : Returns first child tag or text or comment in the accessed tag.
  8. lastChild : Returns last child tag or text or comment in the accessed tag.
  9. nextElementSibling : Returns next sibling of accessed element which is a tag.
  10. previousElementSibling : Returns previous sibling of accessed element which is a tag.
  11. nextSibling : Returns next sibling of accessed element which can be a tag, text or comment.
  12. previousSibling : Returns previous sibling of accessed element which can be a tag, text or comment.
