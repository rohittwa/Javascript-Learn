DOM Create methods

  1. createElement() : Used to create a Tag in HTML by using Javascript.
  2. createText() : Used to write a Text in HTML by using Javascript.
  3. createComment() : Used to write a Comment in HTML by using Javascript.
  4. createDocumentFragment() :
    • The document fragment interface represents a minimal document object that has no parent.
    • A common use for document fragment is to create one, assemble a DOM subtree within it, then append or insert the fragment into the DOM using Node interface methods such as append child() or insertBefore().
    • Document fragment isn't the part of the tree structure, changes made to the fragment don't affect the document.
    • Doing this moves the fragment's nodes into the DOM, leaving behind an empty document fragment.
    • An empty document fragment can be created using the
      document.createDocumentFragment() method.

Methods to insert a text in a tag.

  1. appendChild(child_name) : It appends the text etc. inside the end of the tag.
  2. insertBefore(child_name, target) : It inserts the text etc. before a specified tag.
  3. insertAdjacentElement(position, element_name) : Used to create and insert a tag somewhere in the body.
    Properties in this method :-
    • beforeBegin : Inserts the element before the tag.
    • afterBegin : Inserts the element after start of the tag.
    • beforeEnd : Inserts the element before end of the tag.
    • afterEnd : Inserts the element after end of the tag.
  4. insertAdjacentHTML : Used to create a tag and text and insert it somewhere in the body
    Properties in this method :-
    • beforeBegin : Inserts the element before the tag.
    • afterBegin : Inserts the element after start of the tag.
    • beforeEnd : Inserts the element before end of the tag.
    • afterEnd : Inserts the element after end of the tag.
  5. insertAdjacentText : Used to create a text and insert it somewhere in the body.
    Properties in this method :-
    • beforeBegin : Inserts the element before the tag.
    • afterBegin : Inserts the element after start of the tag.
    • beforeEnd : Inserts the element before end of the tag.
    • afterEnd : Inserts the element after end of the tag.
