DOM Css style methods

Q. DOM Css styling methods.
Ans:
  • style : It is used to get or set the style of an element.
  • className : It is used to get or set the className of an element.
  • classList: It is used to get or set className to an element in the form of array.
    Methods in classList :-
    1. add("className") : It is used to add one or multiple classes in an element.
    2. remove("className") : It is used to remove one or multiple class from an element.
    3. toggle("className") :
    4. contains(className) : Used to find a class in an element. It returns true or false.
    5. item(className) : Used to get the className by specifying index.

    Property in classList :-
    1. length : used to find the number of classes in an element.
