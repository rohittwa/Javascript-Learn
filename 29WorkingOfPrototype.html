Working of prototype

Prototype is the property of the function which points to the prototype object. Prototype object can be accessed using Function_name.prototype.
When you create a new object of that function using new keyword JS engine creates an object and sets a property named __proto__ which points to its function's prototype object.
Ex:-
function Mobile(){

}
var lg = new Mobile();
lg.__proto__ === Mobile.prototype
lg.__proto__.a === Mobile.prototype.a
