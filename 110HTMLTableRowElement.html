HTML Table Row Element

The HTMLTableRowElement interface provides special properties and methods for manipulating the layout and presentation of rows in an HTML table.

Properties:
  • cells : It returns a live HTMLCollection containing the cells in the row. The HTML collection is live and is automatically updated when cells are added or removed.
  • rowIndex
  • sectionRowIndex
Methods:
  • insertCell() : This method inserts a row in the table and returns the reference of the cell.
  • deleteCell()
