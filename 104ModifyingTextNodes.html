Modifying Text Nodes

Properties:
  • Data
  • Length
Methods:
  • appendData(String) : data property is used to get or set the text in a textNode and commentNode.
  • deleteData(start, length) : length property is used to get the length of the text in a textNode and commentNode.
  • insertData(start, string) : It appends the given string at a specified index of a textNode and commentNode.
  • replaceData(start, length, string) : Used to replace the data from a specified position.
  • splitText(start) : Used to split a texNode and commentNode from a specified index to end.
  • substringData(start, length)
