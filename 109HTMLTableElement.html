HTML Table Element

HTMLTableElement interface provides special properties and for manipulating the layout and presentation of tables in an HTML document.

Properties:
  • caption : This property represents the table caption. If no caption element is associated with the table, it can be null.
  • tHead : This property represents the table's <thead> element. Its value can be null if there is no such element.
  • tFoot : This property represents the table's <tfoot> element. Its value can be null if there is no such element.
  • rows : This property returns a live collection of all the rows in the table, including the rows contiaing within any <thead>, <tfoot> and <tbody> elements. Although the property itself is read-only, the returned object is live and allows the modification of the content.
  • tBodies : This read-only property returns a live HTMLCollection of the table bodies. Although the property is read-only, the returned object is live and allows the modification of its content.
Methods:
  • createCaption() : If the caption exists then it returns the caption else it creates the emtpy caption.
  • deleteCaption() : It removes the caption.
  • createTHead() : If the <thead> exists then it returns the <thead> else it creates the emtpy <thead>.
  • deleteTHead() : It removes the thead.
  • createTFoot() : If the tfoot exists then it returns the tfoot else it creates the emtpy tfoot.
  • deleteTFoot() : It removes the tfoot.
  • insertRow() : This method inserts a new row in the table and returns a reference to the new row.
    Syntax: var row = HTMLTableElement.insertRow(index = -1);
    index is the row index of the new row.
    row is assigned a new reference to the new row. A reference to HTMLTableRowElement. If index is -1 or equal to the number of rows, the row is appended as the last row. If index is greater than the number of rows, an IndexSizeError exception will result. If index is omitted the default is -1.
    If a table has multiple body elements, by default, the new row is inserted into the last tbody.
  • deleteRow() : Used to delete a Row.
