CSS Style Declaration Object

The CSSStyleDeclaration object represents a collection of CSS property-value pairs. It is used in API:

  • HTMLElement.style
  • window.getComputedStyle()

Properties:
  • cssText : It is used to give style to an element. It overrides the style which we have given in the style tag.
  • length : It shows the number of style properties defined for the element by using cssText.
  • parentRule
Methods:
  • getPropertyValue(property) : It gives the value of specified css style property.
  • getPropertyPriority(property)
  • removeProperty(property)
  • setProperty(property, value, priority)
  • item(index)
